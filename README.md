# Smart Pharmacy – dawaatech

**Graduation Project**: A smart pharmacy system integrating a web application with a robotic arm to automate medicine dispensing.

## Overview
Smart Pharmacy (dawaatech) automates medicine dispensing inside pharmacies, reducing human error and improving efficiency. It connects doctors, patients, and a robotic arm for safe and accurate delivery.

## Key Features
- Web-based system for doctors and patients
- OTP-based patient verification
- Automated medicine dispensing using robotic arm
- Barcode verification for medicine safety
- Real-time database updates

## Technologies Used
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Hardware:** Arduino R4 WiFi, Tinkerkit Braccio Robot
- **Libraries:** Arduino Servo Library, WiFi Library, PHP Mailer, cURL

## Repository Structure
- `frontend/` – Web application user interface
- `backend/` – Server-side logic, APIs, and Arduino communication
- `hardware/` – Arduino code for robotic arm and barcode scanning
- `database/` – MySQL schema and related files
- `docs/` – Documentation, diagrams, reports
